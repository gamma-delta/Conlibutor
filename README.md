## Using the Library

In Gradle:

```groovy
repositories {
    maven { url 'https://maven.blamejared.com' }
}

dependencies {
    implementation "at.petra-k:conlibutor:conlibutor-${conlibutorVersion}"
}
```

[Check out the latest versions here](https://maven.blamejared.com/at/petra-k/conlibutor/conlibutor/).
